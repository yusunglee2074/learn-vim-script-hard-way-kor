# Echoing Message

첫번째로 살펴볼 것은 `echo`와 `echom` 입니다.

`:help echo`와 같은 명령어로 풀 documentation을 읽을 수 있습니다. 이 책을 읽는 동안에는 항상 새로운 커맨드를 만날때마다 `:help`를 사용해서 조금 더 배우세요.

    :echo "hello, world"

`echo`는 이렇게 사용합니다. 빔 창 아래에 메시지를 볼 수 있습니다.

## 지속성 있는 echoing

이번에는 `echom`을 사용해 보겠습니다.

    :echom "hello, again!"

이 두 커맨드의 다른점을 보려면 아래 커맨드를 입력해보세요.

    :messages

메시지 리스트를 볼 수 있습니다. 여기서에는 첫번째로 입력했던 `hello, world!`를 찾을 수 _없을_ 것입니다. 2번째로 입력한 메시지는 찾을 수 있죠.

추후 빔스크립트를 사용함에 있어 디버깅 하기위해 어떤것을 프린팅 할떄 편하게 사용할 수 있을 것 입니다. `:echo`는 스크립트가 끝나면 사라지지만 `:echom`은 output을 저장하고, `:message`커맨드에서 다시 볼 수 있습니다.


## 코멘트

좀더 가기전에 다른 언어들처럼 주석처리 하는법을 배워보겠습니다. `~/.vimrc`같은 파일을 작성할 `"`기호 뒤에 내용을 적으면 그것은 주석이 됩니다. 예를 들어

    " Make space more useful
    nnoremap <space> za

이것은 _언제나_ 작동하는 것은 아닙니다.(빔스크립트의 X같은 부분입니다.) 하지만 대부분의 경우에 작동합니다. 언제 또 왜 작동안하는지는 나중에 다루겠습니다.
